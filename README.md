# PulseAgent

Modern görev ve zaman yönetimi uygulaması

## Özellikler

- Görev yönetimi
- Zamanlayıcı
- Alt görevler
- Performans takibi
- Modern ve kullanıcı dostu arayüz

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npm start
```

## Geliştirme

```bash
# Android için
npm run android

# iOS için
npm run ios
```
